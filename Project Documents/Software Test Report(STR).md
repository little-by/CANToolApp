# CanToolApp for Windows 测试文档
## 文档概述
本文档记录了在CanToolApp for Windows开发过程中，对开发的软件使用黑盒测试中的等价类划分法进行的单元测试，与预期结果进行了对比，为完善及改进软件的功能提供依据。并且对测试小组的功能测试结果进行了评价。

## 各功能模块单元测试结果概述
